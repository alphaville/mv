Performs matrix-vector multiplication on the device.

Parameters
dAAddress of matrix A on the device in column-major order
dxAddress of vector x on the device
dyAddress of result y = A*x on the device
nRowsNumber of rows of A
nxSize of x (number of columns of A)
Template Parameters
TData type of the matrix and vector elements. The use of float is recommended.
blkThis is the block size which is necessary to statically allocate shared memory. It was found that static allocation offers a significant performance benefit.
